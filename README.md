# Portfolio Satria

Portfolio website untuk Muhammad Satria Rizky, mahasiswa Universitas Tarumanagara yang sedang belajar pengembangan web.

## Deskripsi Proyek

Website portfolio ini dibuat untuk menampilkan profil, proyek, keterampilan, dan informasi kontak. Website ini memiliki desain modern dengan animasi yang halus dan responsif untuk semua perangkat.

## Fitur Utama

- **Hero Section**: Perkenalan diri dengan foto profil dan tombol "Get In Touch"
- **About Section**: Deskripsi singkat tentang diri dan passion dalam pengembangan web
- **Projects Section**: Showcase proyek-proyek yang telah dikerjakan dengan teknologi yang digunakan
- **Skills Section**: Keterampilan teknis dan soft skills yang dimiliki
- **Contact Section**: Formulir kontak dan link ke media sosial (Instagram, LinkedIn, GitHub)
- **Responsive Design**: Kompatibel dengan desktop, tablet, dan mobile
- **Dark Mode Support**: Mendukung tema gelap dan terang
- **Smooth Animations**: Animasi scroll dan hover menggunakan Framer Motion

## Teknologi yang Digunakan

### Frontend Framework
- **Next.js 16.0.1**: React framework untuk production dengan App Router
- **React 19.2.0**: Library JavaScript untuk membangun user interface
- **TypeScript**: Superset JavaScript dengan type safety

### Styling & UI
- **Tailwind CSS 4**: Utility-first CSS framework untuk styling
- **Framer Motion**: Library untuk animasi dan gesture di React

### Development Tools
- **ESLint**: Linting untuk kode JavaScript/TypeScript
- **PostCSS**: Tool untuk transformasi CSS

### Deployment
- **Vercel**: Platform deployment untuk Next.js applications
- **GitHub**: Version control dan hosting repository

## Cara Menjalankan Proyek

### Prerequisites
- Node.js (versi 18 atau lebih baru)
- npm atau yarn

### Instalasi dan Setup

1. **Clone repository**:
   ```bash
   git clone https://github.com/satoriaa/porto-web.git
   cd portfolio-satria
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Jalankan development server**:
   ```bash
   npm run dev
   ```

4. **Buka browser** dan akses [http://localhost:3000](http://localhost:3000)

### Build untuk Production

```bash
npm run build
npm start
```

## Scripts yang Tersedia

- `npm run dev`: Jalankan development server
- `npm run build`: Build aplikasi untuk production
- `npm run start`: Jalankan production server
- `npm run lint`: Jalankan ESLint untuk checking kode

## Deployment

Proyek ini dapat di-deploy ke Vercel dengan mudah:

1. Push kode ke GitHub repository
2. Connect repository ke Vercel
3. Deploy otomatis akan berjalan
